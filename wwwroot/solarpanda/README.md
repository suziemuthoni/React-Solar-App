## How to Use

### Option 1: Direct File Access
Simply open `index.html` in any web browser.

### Option 2: Host with ASP.NET Core
The files are already in your `wwwroot/solarpanda/` folder, which means your ASP.NET Core application can serve them directly.

Access via: `http://localhost:PORT/solarpanda/`

### Option 3: Use a Local Server
```bash
# If you have Python installed:
cd wwwroot/solarpanda
python -m http.server 8000

# Then visit: http://localhost:8000
```

## Customization

All images are currently linked to the original solarpanda.com URLs. If you want to host images locally:

1. Download the images from the source URLs
2. Save them in an `images/` folder
3. Update the `src` attributes in `index.html`
